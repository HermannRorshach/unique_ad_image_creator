Интерфейс:
    - Страница с формой для сущности объявление (стандартный CRUD):
        -- Заголовок объявления
        -- Описание объявления
        -- Категория объявления
        -- Аплодер для картинок
    - Страница для генерации уникальных копий объявления:
        -- Инпут с количеством копий
        -- Инпут с шаблоном копирования
        -- Кнопка "Сгенерировать"
        -- Кнопка "Выгрузить в ексель"



Сервер:
    - Таблицы adverts и similar_adverts

create table if not exists adverts
(
    id              int unsigned auto_incrementprimary key,
    title           varchar(128) default ''  not null,
    category_id     int            default 0    not null,
    description     text null,
    images          text null,
    created_at      timestamp                null,
    updated_at      timestamp                null,
    status_id       smallint     default 1   not null,
    is_deleted      tinyint(1)   default 0   not null
)

create table if not exists similars_adverts
(
    id              int unsigned auto_incrementprimary key,
    advert_id       int            default 0    not null,
    title           varchar(128) default ''  not null,
    category_id     int            default 0    not null,
    description     text null,
    images          text null,
    created_at      timestamp                null,
    updated_at      timestamp                null,
    status_id       smallint     default 1   not null,
    is_deleted      tinyint(1)   default 0   not null
)


Как должно работать:
    - Пользователь заполняет форму создания объявления и сохраняет сущность в табл adverts + картинки в storage
    - Затем переходит на страницу генерации объявлений и жмет "Сгенерировать".По клику отправляется запрос c advert_id, template, copies_count на сервер.
    - Сервер по advert_id находит запись из табл adverts, берет title,description и по шаблону уникализиурет  объявления(взять у Влада шаблон) и режет фото (как резать фото, спросить у Влада)
    - Полученные данные сохранить в  similar_adverts + картинки в storage.
    - По клику на кнлопку "Выгрузить в ексель" выгрузить данные в excel по шаблону для Авито (взять у Влада шаблон)
